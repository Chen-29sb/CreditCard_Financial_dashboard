# CreditCard_Financial_dashboard
This project is a Credit Card Transaction Analysis Dashboard built using data visualization tools to analyze customer behavior, revenue trends, and transaction patterns.
The dashboard provides actionable insights into financial performance, customer segmentation, and operational metrics to support data-driven decision-making.

**Tools & Technologies Used**
Power BI – Data visualization & dashboard creation
MySQL / Excel – Data storage and preprocessing
DAX – Calculations and measures
Data Cleaning & Transformation
